Easy to use Wordpress plugin to insert ReachPPC Link Unit to your pages.

== Description ==

Use this plugin to quickly and easily insert ReachPPC to your pages


== Installation ==

1. Unzip and Upload the folder 'reachppc' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings and configure your account ID in the options page.
